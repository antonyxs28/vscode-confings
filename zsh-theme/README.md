# Zsh Custom Prompt (Amuse-based)

Prompt minimalista e profissional para Zsh, com suporte a:
- Git (branch limpa)
- Node.js (versão automática)
- Horário em tempo real
- Visual clean e dev-friendly

## Preview
![preview](preview.png)

## Requisitos
- Zsh
- Oh My Zsh
- Node.js
- Git

## Uso
1. Copie o conteúdo de `amuse-custom.zsh`
2. Cole no seu `~/.oh-my-zsh/themes/amuse.zsh-theme`
3. Recarregue o terminal:
```bash
exec zsh
